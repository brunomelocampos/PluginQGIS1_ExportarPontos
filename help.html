Ajuda - Exportar Dados Geométricos para TXT

Este plugin permite exportar as coordenadas e atributos de camadas vetoriais (pontos, linhas ou polígonos) para um arquivo de texto tabulado (.txt).

Funcionalidades Principais

Camada de Entrada

Selecione qualquer camada vetorial (pontos, linhas ou polígonos) que você deseja exportar.

Nota: Para linhas e polígonos, o plugin automaticamente:
  • Cria índice espacial para otimização
  • Extrai todos os vértices
  • Remove geometrias duplicadas

Processamento de Dados

  • Apenas Feições selecionadas: Exporta apenas os elementos que estão selecionados na camada.
  • Índice Espacial Automático: Melhora a performance durante o processamento.
  • Transformação de Coordenadas: Converte para o SRC de saída especificado.

Atributos dos Pontos

  • Nome do Ponto: Escolha o campo da tabela de atributos como identificador.
  • Descrição (opcional): Adicione um campo descritivo adicional.
    • Para vértices de linhas/polígonos, o índice do vértice é usado como nome padrão.

Coordenada Z

  • Usar Z da geometria: Disponível apenas para geometrias 3D (PointZ, LineStringZ, etc.)
  • Usar Z de um DEM: Extrai elevação de um raster selecionado.
  • Usar Z de um campo: Utiliza valores numéricos de um campo da tabela.
Atenção: A opção "Usar Z da geometria" será desativada automaticamente se a camada não tiver dimensão Z.

Configurações de Saída

  • SRC de Saída: Define o sistema de coordenadas para o arquivo final.
  • Ordem das Colunas: Escolha entre X/Y ou Y/X.
  • Separador Decimal: Ponto (.) ou Vírgula (,) para valores decimais.
  • Precisão Numérica:
    • Natural: Mantém a precisão original
    • Arredondada: Arredonda para N casas decimais
    • Truncada: Corta para N casas decimais sem arredondar
Dica: O arquivo de saída é formatado com tabulações entre colunas para fácil importação em planilhas.

Arquivo de Saída

Escolha o local e nome do arquivo .txt que será gerado. O formato do arquivo inclui:

Nome[TAB]Descrição[TAB]X[TAB]Y[TAB]Z
Ponto1[TAB]Descrição1[TAB]123.456[TAB]456.789[TAB]100.0
Vértice2[TAB][TAB]124.567[TAB]457.890[TAB]101.5

As colunas Descrição e Z são opcionais e aparecem apenas quando configuradas.

Fluxo de Processamento

  1. Selecionar camada de entrada (pontos, linhas ou polígonos)
  2. Criar índice espacial (automático)
  3. Para linhas/polígonos: extrair vértices e remover duplicados
  4. Transformar coordenadas para o SRC de saída
  5. Aplicar formatação e precisão numérica
  6. Gerar arquivo TXT tabulado
